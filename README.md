Commando Game (minimal HTML5)

Quick start

- Open `index.html` in your browser or run a tiny local server:

```
# Python 3
python -m http.server 8000
# then open http://localhost:8000/CommandoGame/
```

Controls

- Move: Left/Right arrows or A/D
- Shoot: Space
- Restart after death: R

Files

- index.html — main page
- src/game.js — core game logic
- src/styles.css — minimal styles

Notes

This is a minimal single-file game loop using canvas. I can extend with sprites, sounds, levels, and mobile controls if you want.
